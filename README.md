### 영화 리뷰 사이트(Movie Review) -개인 프로젝트

#### 목적
* 영화의 정보를 간략하게 설명하고, 사용자들이 평점을 내리는 사이트를 만들자
* 간단한 게시판 기능을 넣어보자(기본 CRUD, 답변 구현)
* 스프링 시큐리티를 사용해서 로그인, 로그아웃 처리 구현
* AWS를 이용한 DB 관리(RDS), 배포 및 호스팅

#### 개발 환경
> Front-End
* Bootstrap 4 (템플릿 구매 후 사용)
* Jquery 3

> Back-End
* Spring 5.0
* JDK 1.8
* Oracle 11g
* MyBatis 3.4
* Tomcat 8.5
* Maven 2.9
